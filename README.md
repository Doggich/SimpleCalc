# SimpleCalcC
A simple calculator 
in which you can use standard mathematical operations, as well as:
calculate sine, cosine, taganis, cotagens, power and the Euclidean algorithm

Syntax: calc.out/.exe opeartion number1 number2

`
./calc.out add 34 5
39.000000
`

`
./calc.out div 378 4
94.500000
`

`
./calc.out sin 30 0
0.500000
`

`
./calc.out pow 2 30
1073741824
`

`
./calc.exe gcd 45 15
15
`
